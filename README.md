# Vocaplace

Vocaplace is a premium EdTech platform offering a 100% Job Guarantee Digital Marketing Course. Our "Pay After Placement" model allows students to learn SEO, Performance Marketing, and AI Automation directly from industry experts and only pay tuition once they secure a high-paying job.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4, Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Next.js Static Export

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production
To build a static export of the site:
```bash
npm run build
```
The output will be placed in the `out/` directory.
